## 📦 実行方法

Linux の実行方法は特殊です  
1. ダウンロードしたファイルのフォルダに移動 (``cd /home/ユーザー名/Download`` ``cd /home/ユーザー名/ダウンロード`` など)
2. 実行権限を与える (``chmod +x Clock-CLI_Linux*``)
3. 実行する (``./Clock-CLI_Linux*``)
(もしかしたら Mac も同じかも?)
